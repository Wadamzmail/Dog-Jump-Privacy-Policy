سياسة الخصوصية - نطة كلب

نحن في تطبيق "نطة كلب - لعبة سودانية تقليدية" نحترم خصوصيتك ونلتزم بحماية بياناتك. توضح هذه السياسة كيفية جمع البيانات واستخدامها داخل التطبيق.

البيانات التي نجمعها

  • الاسم أو اللقب: يطلب التطبيق من المستخدم إدخال اسم أو لقب لاستخدامه داخل اللعبة، مثلاً في قائمة المتصدرين أو الغرف المتعددة اللاعبين. هذه البيانات لا تُخزن خارج الجهاز.
  • عنوان IP: يتم استخدام عنوان IP فقط عند إنشاء غرفة لعب عبر الواي فاي باستخدام مكتبة KryoNet. لا يتم إرسال هذا العنوان لأي سيرفر خارجي.

الأذونات التي نطلبها ولماذا

  • INTERNET: هذا الإذن مطلوب حتى يتمكن التطبيق من إنشاء اتصال محلي (LAN) بين اللاعبين باستخدام شبكة Wi-Fi. لا يستخدم التطبيق هذا الإذن للاتصال بأي خوادم خارجية.
  • ACCESS_WIFI_STATE (اختياري): يُستخدم فقط للتحقق مما إذا كان الجهاز متصلاً بشبكة Wi-Fi قبل بدء اللعب الجماعي.
  • ACCESS_NETWORK_STATE (اختياري): يُستخدم لتحديد نوع الاتصال بالشبكة (Wi-Fi أو بيانات) لتحسين تجربة المستخدم.

كيف نستخدم البيانات

  • الاسم يستخدم فقط لعرضه داخل اللعبة.
  • عنوان الـ IP يُستخدم مؤقتًا لإنشاء اتصال محلي بين اللاعبين (LAN) ولا يُستخدم لأي أغراض أخرى.

الأمان

نحن لا نقوم بجمع أو تخزين أي بيانات حساسة. لا يتم إرسال أي بيانات لخوادم خارجية، وكل المعلومات تظل داخل الجهاز أو في الاتصال المحلي فقط.

عدم مشاركة البيانات

لا نشارك أي بيانات شخصية مع أي طرف ثالث.

Privacy Policy - Dog Jump

We at "Dog Jump - Traditional Sudanese Game" respect your privacy and are committed to protecting your data. This policy explains what data we collect and how we use it within the app.

Information We Collect

  • Name or Nickname: The app asks the user to enter a name or nickname for display inside the game (such as in leaderboards or multiplayer rooms). This data is not stored externally.
  • IP Address: The IP address is used only when hosting a local Wi-Fi multiplayer room using the KryoNet library. The IP is not collected or sent to any server.

Permissions We Request and Why

  • INTERNET: This permission is required to allow the app to create a local (LAN) connection between players over Wi-Fi. The app does not use this permission to connect to any external servers.
  • ACCESS_WIFI_STATE (optional): Used only to check if the device is connected to a Wi-Fi network before starting multiplayer mode.
  • ACCESS_NETWORK_STATE (optional): Used to detect the type of network connection (Wi-Fi or mobile data) to improve user experience.

How We Use the Data

  • The name is only used for display inside the game.
  • The IP address is used temporarily to allow direct LAN communication between players, and is not used for any other purpose.

Data Security

We do not collect or store any sensitive personal data. No data is sent to external servers. All operations remain on the user's device or through local network communication.

No Data Sharing

We do not share any user data with third parties.
